## :boom: Installation

```bash
# clone this repo
$ git clone git@github.com:LucasPereiraMiranda/lucas-pereira-miranda.github.io.git

# use node 18 with .nvmrc reference
$ nvm use

# install dependencies
$ yarn install

# execute local
$ yarn dev

```
